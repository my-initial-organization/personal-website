Apartment LCE Optimizing Software

Project information

  • Category: Desktop Application
  • Project date: June, 2022
  • Project URL: https://github.com/Gayashan-Dananjaya/Apartment-LCE-LCC-Optimizer

This software tool is developed to optimize the Life Cycle Cost (LCC), the Life Cycle Energy (LCE) and Carbon Footprint by quantitatively analyzing the avaiable construction material alternatives. Further, the user can alter the weightages given for each parameter (LCC, LCE, Carbon footprint)
